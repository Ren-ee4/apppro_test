## gitでブランチを作る

ブランチとは枝のこと、開発において違う系統で行いたい場合はブランチを作ります。
かっこよくいうとブランチを切るといいます。

- 今のブランチを見る

'''
git branch
'''
- ブランチを作成する

'''
git branch branch branch_test
'''
- ブランチを移動

'''
git checkout branch_test
'''
